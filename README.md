## Overview

This project is an interactive web application that presents key members of the Van der Linde gang through a structured carousel interface. Inspired by the narrative depth of *Red Dead Redemption 2*, the application provides a focused exploration of each character, combining visual elements, descriptive text, and controlled navigation.

---

## Objective

The purpose of this project is to apply core front-end development concepts in a practical context, emphasizing clarity of interface, logical state management, and user interaction. It serves as a foundational exercise in building dynamic web components using vanilla JavaScript.

---

## Features

* Sequential navigation between characters
* Dynamic rendering of images, names, and descriptions
* Controlled user flow with conditional button visibility
* Audio feedback to enhance user experience
* Reset functionality integrated into the interface

---

## Technologies

* **HTML5** – structural organization
* **CSS3** – layout and visual styling
* **JavaScript (Vanilla)** – logic, DOM manipulation, and event handling

---

## Architecture

The application is based on a simple state-driven model, where a single index variable controls the displayed content. Arrays store character data, and the interface updates dynamically through DOM manipulation. Navigation logic ensures consistency and prevents invalid transitions.

---

## Usage

1. Open the application in a web browser
2. Click the **Start** button to initialize the carousel
3. Use the navigation controls to browse characters
4. At the end of the sequence, return to the beginning using the reset option

---
